# 遊戲入口網站

一個使用React開發的遊戲入口網站，用於展示和導向我開發的各種遊戲。

## 特點

- 響應式設計，適配不同設備
- 精美的UI/UX設計
- 展示遊戲預覽和描述
- 直接連接到各遊戲GitHub頁面

## 包含遊戲

- 黑白棋/奧賽羅 (React Reversi Game)
- 試管倒球遊戲 (Ball Sort Puzzle)
- 自走棋對戰 (Auto Chess Battle)
- 撲克遊戲 (Poker Game)

## 技術棧

- React
- React Router (HashRouter)
- Tailwind CSS
- Framer Motion

## 開發指南

1. 克隆此存儲庫
   ```
   git clone https://github.com/yanchen184/game-portal.git
   cd game-portal
   ```

2. 安裝依賴
   ```
   npm install
   ```

3. 啟動開發伺服器
   ```
   npm start
   ```

4. 構建生產版本
   ```
   npm run build
   ```

5. 部署到 GitHub Pages
   ```
   npm run deploy
   ```
