# 遊戲入口網站

一個使用React開發的遊戲入口網站，用於展示和導向我開發的各種遊戲。

## 特點

- 響應式設計，適配不同設備
- 精美的UI/UX設計
- 展示遊戲預覽和描述
- 直接連接到各遊戲GitHub頁面

## 包含遊戲

- 黑白棋/奧賽羅 (React Reversi Game)
- 試管倒球遊戲 (Ball Sort Puzzle)
- 自走棋對戰 (Auto Chess Battle)
- 撲克遊戲 (Poker Game)

## 技術棧

- React
- React Router
- Tailwind CSS
- Framer Motion
